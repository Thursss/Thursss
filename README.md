# **黑羊**

  从前有个国家，里面人人是贼。

  黑羊，体格中等，体躯匀称，略呈长方形。头呈三角形，鼻梁平直，两耳向前倾立，公、母羊绝大多数有角、有髯，公羊角粗大，呈镰刀状，略向后外侧扭转，母羊角较小，多向后上方弯曲，向外侧扭转。毛被光泽好，大多为黑色，少数为白色、黄色和杂色。毛被内层生长有短而稀的绒毛。
